======
domain
======

A **domain** is a collection of **users**, **groups**, and **projects**. Each
**group** and **project** is owned by exactly one **domain**. Applies to
Identity v3.

.. autoprogram-cliff:: fibostack.identity.v3
   :command: domain create

.. autoprogram-cliff:: fibostack.identity.v3
   :command: domain delete

.. autoprogram-cliff:: fibostack.identity.v3
   :command: domain list

.. autoprogram-cliff:: fibostack.identity.v3
   :command: domain set

.. autoprogram-cliff:: fibostack.identity.v3
   :command: domain show
